Quickstart: Build X and y from Fivetran/dbt tables

Use the new data loaders to go from standardized warehouse tables to model-ready inputs in a few lines:

from pymc_marketing.data.fivetran import (
    process_fivetran_ad_reporting,
    process_fivetran_shopify_orders_unique_orders,
)
import pandas as pd

# Query standardized tables from your warehouse
campaign_df = pd.read_sql("SELECT * FROM schema.ad_reporting_ad_report", con)
orders_df = pd.read_sql("SELECT * FROM schema.shopify_report", con)

# Create design matrix (X) and target (y)
X = process_fivetran_ad_reporting(campaign_df, value_columns="spend", rename_date_to="date")
y = process_fivetran_shopify_orders_unique_orders(orders_df)

# Fit your MMM
